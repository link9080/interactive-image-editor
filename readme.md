# Interactive Image Editor 🎨

直感的でインタラクティブな操作が可能な、ブラウザ完結型の画像編集Webアプリケーションです。

---

## 🚀 主な機能 (Key Features)

* **簡単アップロード**: ドラッグ＆ドロップまたはファイル選択で瞬時に画像を読み込み。
* **多彩なフィルター効果**: サイドバーから好みのフィルターを選択可能。
* **リアルタイム調整**: スライダーを動かすだけで、フィルターの強度をリアルタイムに反映。
* **ワンクリック保存**: 編集が終わった画像は、最高画質のままローカルにダウンロード可能。

---

## 🛠 開発環境のセットアップ (Getting Started)

ローカル環境でプロジェクトを起動するには、以下の手順に従ってください。

### 前提条件 (Prerequisites)

* **Node.js** (v18以上を推奨)
* **npm** または **yarn** / **pnpm**

### インストール (Installation)

1. リポジトリをクローンします。
   ```bash
   git clone [https://github.com/ユーザー名/interactive-image-editor.git](https://github.com/ユーザー名/interactive-image-editor.git)
   cd interactive-image-editor
2. nodeモジュールをインストール
 ```bash
 npm install
 ```
3. サーバ起動
```bash
npm run dev
```
